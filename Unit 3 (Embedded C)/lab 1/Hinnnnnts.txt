app.c ,uart.c, uart.h are written.
app.o & uart.o are extracted from app.c & uart.c.
          arm-none-eabi-gcc.exe -c -g -I . -mcpu=arm926ej-s app.c -o app.o
          arm-none-eabi-gcc.exe -c -g -I . -mcpu=arm926ej-s uart.c -o uart.o
a startup.s is written.
startup.o is extracted from startup.s.
          arm-none-eabi-as.exe -mcpu=arm926ej-s startup.s -o startup.o
linker_script.ld is written.

learn-in-depth.elf file is generated by linking these files
          arm-none-eabi-ld.exe -T linker_script.ld startup.o app.o uart.o -o learn-in-depth.elf

learn-in-depth.bin file is generated
          arm-none-eabi-objcopy.exe -O binary learn-in-depth.elf learn-in-depth.bin

learn-in-depth.bin file is burned on qemu
          & 'C:\Program Files (x86)\qemu\qemu-system-arm' -M versatilepb -m 128M -nographic -kernel learn-in-depth.bin
